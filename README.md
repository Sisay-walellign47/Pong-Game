## 📖 About

Python Pong is a simple recreation of the legendary Pong arcade game using Python's built-in Turtle Graphics library.

Two players compete by controlling paddles on opposite sides of the screen. The objective is to prevent the ball from passing your paddle while scoring points against your opponent.

---

## ✨ Features

- 🎮 Two-player gameplay
- 🏓 Ball collision physics
- 📊 Live score tracking
- ⌨️ Keyboard controls
- 🎨 Colorful Turtle Graphics interface
- ⚡ Lightweight and beginner-friendly

---

## 🎮 Controls

### Player 1 (Left Paddle)

| Key | Action |
|------|---------|
| `Q` | Move Up |
| `A` | Move Down |

### Player 2 (Right Paddle)

| Key | Action |
|------|---------|
| `↑` | Move Up |
| `↓` | Move Down |

## 🚀 Getting Started

### Requirements

- Python 3.x

No additional packages are required because the `turtle` module comes pre-installed with Python.

### Run the Game

Clone the repository:

```bash
git clone https://github.com/your-username/python-pong.git
```

Navigate to the project directory:

```bash
cd python-pong
```

Run the game:

```bash
python Pong.py
```

## 🎯 How It Works

### Ball Movement

- The ball moves continuously across the screen.
- It bounces off the top and bottom walls.
- It reverses direction when hitting a paddle.

### Scoring System

- If the ball passes the right paddle, Player 1 scores.
- If the ball passes the left paddle, Player 2 scores.
- After a point is scored, the ball returns to the center.
